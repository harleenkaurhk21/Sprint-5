# React Kanban Board

A Trello-style Task Management Board built using **React.js** and **Vite**. The application allows users to manage tasks by adding, editing, moving, deleting, and searching tasks. Task data is stored using browser Local Storage, so it persists even after refreshing the page.

## 🚀 Live Demo

https://sprint-5-three.vercel.app/

## ✨ Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Move tasks between columns
- Search tasks
- Task priority (High, Medium, Low)
- Local Storage persistence
- Responsive UI

## 🛠️ Technologies Used

- React.js
- Vite
- JavaScript (ES6+)
- CSS3
- HTML5
- Local Storage
